# wx-vcpkg-cmake-test

This is a simple test / example of compiling the "minimal" WxWidgets example, using cmake and vcpkg.

## clone

`git clone --recurse-submodules https://github.com/gnoling/wx-vcpkg-cmake-test.git`

## build

`cd wx-vcpkg-cmake-test
mkdir build
cd build
cmake ..
`

## run

`./minimal`
